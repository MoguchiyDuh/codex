# OOP — Course Roadmap

Language-agnostic. Examples drawn from Python, C++, Rust where relevant.
Focus: concepts and trade-offs, not syntax.

---

## Phases

### Phase 1 — OOP Pillars
Topics: encapsulation, abstraction, inheritance, polymorphism — precise definitions,
not textbook slogans. What each solves, what each costs.

**Exam status:** not taken
**Grade:** —

---

### Phase 2 — Interfaces & Abstract Classes
Topics: interface vs abstract class distinction, programming to interfaces,
duck typing (Python) vs structural typing (Rust traits) vs nominal typing (Java/C++),
when to use each.

**Exam status:** not taken
**Grade:** —

---

### Phase 3 — Composition vs Inheritance
Topics: fragile base class problem, diamond problem, mixin patterns,
composition-over-inheritance in practice, Rust's answer (no inheritance).

**Exam status:** not taken
**Grade:** —

---

### Phase 4 — SOLID Principles
Topics: SRP, OCP, LSP, ISP, DIP — each with a concrete violation example
and a corrected example. Common misapplications.

**Exam status:** not taken
**Grade:** —

---

### Phase 5 — Design Patterns
Topics: Creational (Factory, Builder, Singleton and why it's controversial),
Structural (Adapter, Decorator, Facade), Behavioral (Observer, Strategy, Command).
Pattern vs anti-pattern line.

**Exam status:** not taken
**Grade:** —

---

## Completion

All 5 phases passed (grade ≥ C).
