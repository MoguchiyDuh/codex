# Searching

> Finding an element in a collection.

## Linear search — O(n)

### When it's the only option

## Binary search — O(log n)

### Precondition: sorted input

### Implementation — the off-by-one trap

### Variants: find first, find last, lower bound, upper bound

## Search in a BST — O(log n) average

## Hash table lookup — O(1) average

## Comparison

| Algorithm | Time | Precondition |
|---|---|---|
| Linear | O(n) | None |
| Binary | O(log n) | Sorted |
| BST | O(log n) avg | Tree built |
| Hash | O(1) avg | Hash function |
