# Heap

> A complete binary tree where every parent satisfies the heap property — used as a priority queue.

## Heap property

### Max-heap: parent ≥ children

### Min-heap: parent ≤ children

## Array representation

### Index formulas: parent, left child, right child

## Operations

### Insert — O(log n), sift up

### Extract min/max — O(log n), sift down

### Peek — O(1)

### Heapify — O(n), build from unsorted array

## Heap sort — O(n log n), in-place
