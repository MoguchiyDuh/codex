# Computer Architecture — Course Roadmap

From logic gates to CPU execution. Connects directly to C and systems programming.

---

## Phases

### Phase 1 — Boolean Algebra & Logic Gates
Topics: truth tables, De Morgan's laws, NAND/NOR universality,
sum of products, Karnaugh maps (2–4 variable), logic minimization.

**Exam status:** not taken
**Grade:** —

---

### Phase 2 — Number Systems & Data Representation
Topics: binary, hex, two's complement, sign-magnitude vs two's complement,
overflow in hardware, IEEE 754 float representation, NaN / Inf / denormals.

**Exam status:** not taken
**Grade:** —

---

### Phase 3 — Combinational Circuits
Topics: adders (half, full, ripple carry, carry-lookahead), multiplexers,
decoders, ALU design, propagation delay.

**Exam status:** not taken
**Grade:** —

---

### Phase 4 — Sequential Circuits & State
Topics: latches, flip-flops (D, JK, T), registers, counters,
finite state machines (Mealy vs Moore), clock and timing.

**Exam status:** not taken
**Grade:** —

---

### Phase 5 — ISA & Assembly Concepts
Topics: instruction set architecture, RISC vs CISC, registers vs memory,
addressing modes, calling conventions, stack frames, x86-64 basics.

**Exam status:** not taken
**Grade:** —

---

### Phase 6 — CPU Architecture
Topics: fetch-decode-execute cycle, control unit, datapath,
single-cycle vs multi-cycle, microcode.

**Exam status:** not taken
**Grade:** —

---

### Phase 7 — Pipelining
Topics: pipeline stages, hazards (data, control, structural),
forwarding, branch prediction, out-of-order execution concept.

**Exam status:** not taken
**Grade:** —

---

### Phase 8 — Memory Hierarchy
Topics: SRAM vs DRAM, cache levels (L1/L2/L3), cache lines, associativity,
write policies (write-back vs write-through), TLB, virtual vs physical addresses.

**Exam status:** not taken
**Grade:** —

---

## Completion

All 8 phases passed (grade ≥ C).
